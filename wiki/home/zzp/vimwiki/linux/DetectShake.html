A demo is from [http://my.oschina.net/u/247549/blog/60479 here]

Every diffTime, detect the value of the accelarator, if the value of the accelarator of the last time is x_last,y_last,z_last, and the current value is x,y,z. Then the shake speed is :

speed = (x+y+z-x_last-y_last-z_last)/diffTime

or 

    distance=sqrt(x^2 + y^2 + z^2)
    distance_last=sqrt(x_last^2 + y_last^2 + z_last^2)
    speed = (distance - distance_last)/diffTime
