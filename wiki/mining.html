通过freeshell挖矿

  • 在freeshell

上注册一个帐号.

  • 在这里

管理你的shell

  • 使用ssh 登陆你的freeshell.
    ssh -R9999:localhost:22 -p <port> root@<host>
    for my shell
    ssh -R9999:localhost:22 -p 10189 root@114.214.197.8
  • 在freeshell中反向连接自己的主机做代理。
    ssh -p 9999 -D8888 -f -N tinyfox@localhost
    8888为代理的端口。
  • 使用cpuminer开始挖矿
    ./minerd -x socks5://127.0.0.1:8888 --url http://mine.pool-x.eu/ --userpass tinyfox266.1:x
    -x 参数设置代理

使用gpu, cgminer 挖矿

windows, amd/ati 显卡

  • 在这里下载amd的驱动
  • 点击安装驱动
  • 在这里下载cgminer的windows版本。
  • 在这里。 查看相应cgminer的参数。
  • 重启计算机，使驱动生效。
  • 运行 cgminer -n 来查看显卡的驱动是否已经安装好。
  • 使用cgminer开始挖矿。

linux, amd/ati 显卡

  • 在这里下载amd的驱动，假设下载目录为~/Downloads
  • 卸载linux原有的amd显卡驱动，可以在ubuntu software center中查看
  • 使用终端登陆(<Ctrl+Alt+F1>)
  • 运行 sudo /etc/init.d/lightdm stop,关闭桌面系统。
  • cd ~/Downloads
  • 解压下载的驱动文件，在终端中运行。
    sudo ./amd-catalyst-13.4-linux-x86.x86_64.run
  • 安装好驱动后重启系统。
  • 在这里下载cgminer的linux版本。
  • 在这里。 查看相应cgminer的参数。
  • 运行 cgminer -n 来查看显卡的驱动是否已经安装好。
  • 运行cgminer开始挖矿
