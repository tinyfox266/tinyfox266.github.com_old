Upgrade from the supported version

upgrade to the LTS version

  • sudo apt-get install update-manager-core
  • do-release-Upgrade
And then follow the instructions.

Upgrade to the any version

  • backup the config file /etc/update-manager/release-upgrades
  • open the config file
    sudo vi /etc/update-manager/release-upgrades
  • find the line that reads:
    Promt=LTS
  • change it to say:
    Promt=normal
  • update to the next release
    sudo do-release-upgrade

Upgrade from the unsupported version

  • Download the version of Ubuntu you need to upgrade to(for me it was 9.04 Jaunty Jackalope)
    wget http://old-releases.ubuntu.com/releases/jaunty/ubuntu-9.04-server-i386.iso
  • Make a backup of your current sources.list
    sudo mv /etc/apt/sources.list /etc/apt/sources.bak
  • Mount the ISO so that it can read from
    sudo mount -o loop Ubuntu-9-04-server-i386.iso /media/cdrom
  • Create a new sources.list with the ISO as the source for the Upgrade
    vim /etc/apt/sources.list
    add the following line
    deb file::///media/cdrom <release-version> main restricted
  • Now update your sources and upgrade
    apt-get update
    apt-get dist-upgrade

Once the packages are downloaded, go through the installation and reboot the server once it is done. If the version you have upgraded to has no mirrors available, then repeat the above process.

If you have upgraded to a version with mirrors available, then rename the /etc/apt/sources.list to /etc/apt/sources.iso and /etc/apt/sources.bak to /etc/apt/sources.list

Now you can run #do-release-upgrade to upgrade to the next version, reboot when complete and then repeat this process until you reach either a supported LTS or the most recent version of Ubuntu.
